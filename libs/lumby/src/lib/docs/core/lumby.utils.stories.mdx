## Lumby Utils

Utils have one function so far: `canvasStylesheet`. <br/>
This function is a behemoth that translates props sent from Lumby components and
turns them into an object of callbacks corresponding to specific CSS attributes
represented in object notation. These callbacks can be then called from
any component and they take optional argument that will be taken into account
when new stylesheet is created.

#### - `canvasStylesheet`

```typescript

  const placeItems = (axis, n) => ...
  const spacing = (on, newSize) => ...
  const boxShadow = (elevation) => ...
  const borderRadius = (corners) => ...
  const variant = (status, style) => ...

  return {
    width,
    height,
    cursor,
    display,
    fontSize,
    boxShadow,
    borderRadius,
    alignItems: (n) => placeItems('y', n),
    alignContent: (n) => placeItems('x', n),
    justifyContent: (n) => placeItems('x', n),
    margin: (size) => spacing('margins', size),
    color: (status) => variant(status, 'color'),
    padding: (size) => spacing('paddings', size),
    borderColor: (status) => variant(status, 'borderColor'),
    backgroundColor: (status) => variant(status, 'backgroundColor'),
  };
};
```

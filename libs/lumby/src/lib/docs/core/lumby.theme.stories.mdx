## Lumby Theme

Theming in Lumby is based on class `LumbyTheme` and function `lumbyTheme`.

#### - `LumbyTheme`

`LumbyTheme` is a class. It holds default information regarding various variables used to style Lumby components.
The idea behind `LumbyTheme` is to provide the most commonly used CSS attributes and bind them across all components.

`LumbyTheme` provides a bunch of constants that enforce uniform styling across all components.
`isDark`, `size`, `colors`, `decorators` and `variants` are required by Lumby internally, so unless you provide your own overwrites,
Lumby's default styling <strong>will take over</strong>.

```typescript
export class LumbyTheme {
  isDark = true;
  size = {
    spacing: { xxs: 4, xs: 8, sm: 12, md: 16, lg: 24, xl: 48, xxl: 96 },
    fontSize: { xxs: ... },
    borderWidth: { xxs: ... },
  };
  colors = {
    plain: 'transparent',
    default: '#f5f5f5',
    disabled: '#e0e0e0',
    dark: '#142c44',
    light: '#f5f5f5',
    accent: '#ecac6c',
    primary: '#332857',
    secondary: '#a49ccc',
    ternary: '#a49',
    error: '#c42c34',
    select: (color) => this.colors[color],
  };
  decorators = {
    elevation: { 0: 'none', 1: '...', 2: '...'},
    borderRadius: { xxs: ... }
  }
  variants = {
    default: {
      default: {
        color: this.colors.light,
        borderColor: this.colors.accent,
        backgroundColor: this.colors.dark
      },
      error: { color: ... },
      focus: { color: ... },
      hover: { color: ... },
      active: { color: ... },
      working: { color: ... },
      disabled: { color: ... },
    },
    plain: { default, error, focus ... },
    primary: { default, error, focus ... },
    secondary: { default, error, focus ... },
  }
}
```

#### - `lumbyTheme`

`lumbyTheme` is used to sync all theme sources. It uses [deepmerge](https://www.npmjs.com/package/deepmerge) to
ensure everythig's merged properly.

```typescript
import { ExternalConfig } from '../../external';

const merge = <T extends LumbyTheme>(a: T, b: Partial<T>) => deepmerge<T>(a, b);

export function lumbyTheme(newTheme?: Partial<LumbyTheme>) {
  const theme = new LumbyTheme();

  // if lumby.config.json exists in folder root - merge it with the rest
  const externalTheme = Object.create(ExternalConfig.theme) as LumbyTheme;

  // it is done like so for readability ;)
  if (!externalTheme && newTheme) return merge(theme, newTheme);
  if (externalTheme && !newTheme) return merge(theme, externalTheme);
  if (externalTheme && newTheme) return merge(theme, merge(externalTheme, newTheme));
  return theme;
}
```

## useLumbyTheme

This hook uses `zustand` to react to anything that happens to the theme and inform Lumby's components
about these changes. It also uses [use-dark-mode](https://github.com/donavon/use-dark-mode) plugin to synchronize toggling between themes. <br/>
`useLumbyTheme` takes one optional argument `newTheme`, which should be the theme object.
This `newTheme` argument takes precedence over the default theme and overwrites whatever has been previously set up.

```typescript
import { useLumbyTheme } from '@devutnia/lumby/core';

const { theme, toggleTheme, setTheme, darkMode } = useLumbyTheme(newTheme?: Partial<LumbyTheme>);

console.log(theme); // default theme object
console.log(toggleTheme(differentTheme)); // toggles themes using useDarkMode hook
console.log(setTheme(differentTheme)); // sets new theme
console.log(theme); // differentTheme object
console.log(darkMode); // original object returned by useDarkMode hook
```
